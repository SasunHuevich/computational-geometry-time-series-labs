# Лабораторная работа №1: Дискретное преобразование Фурье (ДПФ)

**Краткое описание**  
В этой работе я реализовал прямое и обратное дискретное преобразование Фурье на языке Rust, проанализировал частотный состав искусственного сигнала и убедиился в обратимости преобразования.

## Что такое ДПФ и зачем оно нужно?

Дискретное преобразование Фурье – это математический инструмент, который переводит сигнал из временно́й области (массив отсчётов) в частотную область (спектр). Спектр показывает, из каких синусоидальных колебаний (гармоник) состоит исходный сигнал и какова их амплитуда и фаза. Это основа для:

- анализа спектра звука, вибраций, радиосигналов;
- фильтрации шумов;
- сжатия данных (JPEG, MP3);
- решения дифференциальных уравнений;
- обработки изображений и многих других областей.

## Как работает программа

Программа выполняет следующие шаги:

1. **Генерация тестового сигнала**  
   Создаётся массив из `N` отсчётов, представляющий сумму двух синусоид с заданными частотами и амплитудами (например, 10 Гц и 25 Гц).

2. **Вычисление прямого ДПФ**  
   Реализована «наивная» версия ДПФ (сложность O(N²)) – она строго соответствует математическому определению.

3. **Анализ амплитудного спектра**  
   Из комплексных коэффициентов спектра вычисляются модули (амплитуды). Программа находит локальные максимумы (пики) в области положительных частот (от 1 до N/2) и определяет соответствующие им реальные частоты (в Гц) по формуле `f = k * fs / N`.

4. **Нормировка амплитуд**  
   Амплитуды пиков нормируются на `N/2`, чтобы их значения соответствовали реальным амплитудам синусоид в исходном сигнале (при условии, что частота точно попадает в бин).

5. **Обратное ДПФ**  
   По полученному спектру восстанавливается сигнал во временной области. Это демонстрирует обратимость преобразования.

6. **Оценка точности**  
   Вычисляется среднеквадратичная ошибка (MSE) между исходным и восстановленным сигналом. Значение порядка 10⁻²⁹ подтверждает, что преобразование работает без потерь (в пределах машинной точности).

## Параметры (настраиваются в `main()`)

| Параметр | Значение по умолчанию | Смысл |
|----------|----------------------|-------|
| `n`      | 64                   | Количество отсчётов сигнала. Влияет на частотное разрешение: `fs / n`. |
| `fs`     | 100.0 Гц             | Частота дискретизации (сколько отсчётов в секунду). Определяет максимальную анализируемую частоту (`fs/2`). |
| `expected` | `[(10.0/fs, 1.0), (25.0/fs, 0.5)]` | Список синусоид в формате `(относительная_частота, амплитуда)`. Относительная частота = физическая частота / fs. |
| `threshold` | `0.1`               | Порог для поиска пиков (игнорируются маленькие значения). |
