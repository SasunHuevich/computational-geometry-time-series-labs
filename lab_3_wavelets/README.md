# Лабораторная работа №3: Вейвлет-преобразование Хаара и чирплет-анализ сигналов

## Краткое описание

Практическая реализация одномерного дискретного вейвлет-преобразования Хаара и чирплет-преобразования на языке Rust. Программа демонстрирует разложение сигнала на низко- и высокочастотные составляющие, последующее восстановление исходного сигнала без потерь, а также анализ сигнала с линейно изменяющейся частотой при помощи чирплетов.

---

## 🔬 Что такое вейвлеты и чирплеты?

### Вейвлет-преобразование

Вейвлет-преобразование представляет собой метод анализа сигналов, позволяющий одновременно определить как частотный состав сигнала, так и момент времени, в который появляются различные компоненты.

В отличие от преобразования Фурье, которое использует бесконечные синусоиды, вейвлет-анализ использует короткие локализованные функции — **вейвлеты**, которые можно:

- сжимать и растягивать (масштаб);
- сдвигать по времени;
- анализировать локальные особенности сигнала.

Одним из самых простых является **вейвлет Хаара**, основанный на попарочном усреднении и вычислении разностей между соседними отсчётами.

---

### Чирплет-преобразование

Чирплет является развитием идеи вейвлетов.

Если обычный вейвлет хорошо обнаруживает участки сигнала с практически постоянной частотой, то чирплет способен анализировать сигналы, частота которых изменяется со временем.

Такие сигналы называются **чирп-сигналами** и широко применяются в:

- радиолокации;
- сонарных системах;
- цифровой связи;
- обработке речи;
- биомедицинских исследованиях.

В программе используется гауссов чирплет с квадратичной фазой, позволяющий эффективно находить участки сигнала с возрастающей или убывающей частотой.

---

## ⚙️ Как работает программа

Программа состоит из двух независимых частей.

### 1. Вейвлет-преобразование Хаара

На первом этапе создаётся простой тестовый сигнал

После этого выполняются следующие действия:

1. Прямое дискретное вейвлет-преобразование Хаара.
2. Разложение сигнала на несколько уровней приближений и деталей.
3. Обратное преобразование.
4. Сравнение восстановленного сигнала с исходным.
5. Вычисление ошибок восстановления:
   - MSE (Mean Squared Error);
   - RMSE (Root Mean Squared Error).

Поскольку преобразование Хаара является обратимым, восстановленный сигнал практически полностью совпадает с исходным, а возникающая ошибка обусловлена только конечной точностью вычислений с плавающей запятой.

---

### 2. Чирплет-анализ

Во второй части программы генерируется линейный чирп-сигнал.

Его мгновенная частота изменяется от

- **5 Гц**

до

- **20 Гц**

за

- **4 секунды**.

После генерации сигнала выполняется перебор множества чирплетов с различными параметрами:

- масштаб;
- временной сдвиг;
- параметр изменения частоты β.

Для каждого набора параметров вычисляется коэффициент корреляции между сигналом и соответствующим чирплетом.

Максимальный коэффициент показывает, какой чирплет наиболее точно соответствует анализируемому сигналу.

Полученные параметры позволяют оценить:

- временное положение участка сигнала;
- характер изменения частоты;
- наиболее подходящий масштаб анализирующей функции.

## 🛠 Параметры системы (Конфигурация в `main()`)

### Вейвлет-анализ

| Параметр | Значение | Назначение |
| :--- | :--- | :--- |
| signal | `[1,1,1,1,0,0,0,0]` | Исходный тестовый сигнал. |
| haar_dwt() | — | Выполняет прямое вейвлет-преобразование Хаара. |
| haar_idwt() | — | Выполняет обратное восстановление сигнала. |

---

### Чирплет-анализ

| Параметр | Значение | Назначение |
| :--- | :--- | :--- |
| n | 256 | Количество отсчётов сигнала. |
| duration | 4.0 с | Продолжительность сигнала. |
| Начальная частота | 5 Гц | Начало линейного изменения частоты. |
| Конечная частота | 20 Гц | Конец линейного изменения частоты. |
| scales | `[0.5, 1.0, 2.0, 4.0]` | Масштабы анализирующих чирплетов. |
| shifts | каждые 10 отсчётов | Возможные временные положения чирплета. |
| omega0 | `2π·10` | Центральная несущая частота чирплета (10 Гц). |
| beta_list | `[-5.0, 0.0, 5.0]` | Возможные скорости изменения частоты. |
